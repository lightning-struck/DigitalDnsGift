## NEXT.JS v.15

## Node version: `20.12.1`

- если nvm не установлен - https://github.com/coreybutler/nvm-windows

- если nvm установлен:

- `nvm install 20.12.1`
- `nvm use 20.12.1`

- `npm install` - установка зависимостей
- `npm run dev` - запустить проект локально в режиме dev
- `npm run build` - запустить билд
